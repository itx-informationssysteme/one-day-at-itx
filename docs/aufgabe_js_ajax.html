1 Aufgaben

 

1.1 Beschreibung

Es soll eine HTML Seite erstellt werden, auf der eine Liste mit Produkten ausgegeben wird. Oberhalb der Produktliste wird ein Filter platziert, über den der Besucher die Liste der Produkte reduzieren kann.

Alle Änderungen am Inhalt der Seite sollen per Javascript bzw. Ajax erfolgen. Das Layout spielt bei dieser Aufgabe keine Rolle.

Als Javacript Hilfsmittel können die folgenden Bibliotheken eingesetzt werden - falls sinnvoll:

  • jQuery
  • Bootstrap

1.2 Details zu Produkten

Produkte bestehen aus den Attributen: Id, Name, Beschreibung. Zu jedem Produkt wird genau 1 Farbe und genau 1 Gewicht hinterlegt.

Als Farbe gibt es die Werte "rot" und "grün". Als Gewicht gibt es die Werte "5 kg" und "10 kg".

Die Produkte werden bei dieser Aufgabe nicht aus einer Datenbank geladen, sondern in PHP durch eine Hilfsfunktion bereitgestellt. Diese ist bereits vorbereitet.

Die Filterwerte und die Produktliste sollen über Ajax geladen werden, wobei die Produktliste vom gewählten Filter abhängt.

1.3 Details zum Filter

 Der Filter ermöglicht das Reduzieren der Listeneinträge über die Auswahl einer Farbe und/oder die Auswahl eines Gewichts.

Farbe: rot | grün

Gewicht: 5 kg | 10 kg

Es kann jeweils nur 1 Wert pro Attribut gewählt werden. Der Filter soll initial nur die Attribute zur Auswahl ermöglichen, die bei den Produkten vorkommen.

1.4 Das bedeutet folgenden Ablauf:

  1. Filter zeigt sich vollständig an
  2. Produktliste zeigt sich vollständig an
  3. Besucher wählt einen Wert im Filter
  4. Produktliste reduziert sich
  5. Bei einer Filterkombination, für die es keine Produkte gibt, wird eine entsprechende Meldung angezeigt
