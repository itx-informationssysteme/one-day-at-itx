1 Beschreibung

Die folgende Website Navigation soll in einer Datenbank gespeichert werden und in verschiedenen Kanälen ausgegeben werden:

Hauptnavigation

                Unternehmen

                               Abteilungen

                               Jobs & Karriere

                Produkte

                               Sensoren

                                               Lichtschranken

                                                               Gabellichtschranken

                                                               Ringlichtschranken

                                                               Rahmenlichtschranken

                                               Bewegungssensoren

                               Zubehör

                               Kabel

                Branchen

Metanavigation

2 Aufgaben

2.1 Datenmodell erstellen

Erstellen Sie ein Datenmodell, mit dem man diese Navigation in einer relationalen Datenbank speichern kann.

2.2 Codierung

Erstellen Sie eine kleine PHP Lösung, um die Beispieldaten als verschachtelte HTML-Liste auszugeben.

Es soll möglich sein, über ein Formular einen beliebigen Knoten (ID) als „Mutterknoten“ für die Darstellung festzulegen. Es werden nur alle „Kindknoten“ des gewählten „Mutterknoten“ ausgegeben.

Wird kein „Mutterknoten“ angegeben, so wird der komplette Navigations-Baum ausgegeben.

Zu jedem Knoten sollen bei der Ausgabe folgende Informationen zur Verfügung stehen:

  • Bezeichner
  • Nummer der Ebene (Start bei 1 unterhalb des Mutterknoten)
  • Gesamt Anzahl der Kindknoten
